## 视觉风格规范

### 1-bit美学原则

```
色彩系统：
┌─────────────────┐
│  ████  纯黑     │
│  ░░░░  抖动黑绿    │
│  ▒▒▒▒  图案黑绿    │
│  ▓▓▓▓  点阵黑绿    │
│  ████  纯绿（背景）│
└─────────────────┘

禁止：抗锯齿、半透明、渐变
推崇：硬边缘、抖动图案、负空间
```

### 屏幕规格

| 参数 | 规格 |
|-----|------|
| 分辨率 | 适配设备分辨率  |
| 色彩深度 | 1-bit（纯黑/纯绿） |
| 刷新率 | 30 FPS（复古感）或 50 FPS（流畅） |
| 安全边距 | 8px（避免屏幕圆角裁切） |

## 技术实现要点

### 精灵优化

```lua
-- 使用位图掩码实现1-bit透明
登月舱精灵 = {
    图像 = "lander.png",  -- 纯黑白，无alpha
    掩码 = "lander_mask.png", -- 1=可见, 0=透明
    中心点 = {8, 8}  -- 旋转中心
}

-- 动态生成地形
函数 生成地形(种子)
    噪声 = perlin噪声(种子)
    对于 x = 0, 400, 2 做  -- 每2像素一个点
        高度 = 120 + 噪声(x*0.01) * 60
        添加地形点(x, 高度)
    结束
结束
```


```

### 碰撞检测

```lua
-- 简化碰撞：登月舱底部4个点
着陆点 = {
    {x-6, y+8}, {x-2, y+8},
    {x+2, y+8}, {x+6, y+8}
}

函数 检查着陆()
    对于 每个 点 在 着陆点 做
        如果 点.y > 地形高度(点.x) 则
            返回 真
        结束
    结束
    返回 假
结束
```
